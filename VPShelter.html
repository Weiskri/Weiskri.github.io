Virtual Pet Shelter

Skills used: C#, OOP, .NET

This console application game builds upon my virtual pet project. With this game, however, the user is in charge of a shelter full of rabbits with varying personalities. While much of the logic is the same as the virtual pet project, each rabbit has its own likes and dislikes based on its personality, which makes the game more fun and challenging. So, for example, when the user tries to feed Nietzsche a treat, the rabbit will always refuse the food and bite. Others will only accept certain treats.
